# Mobile Evidence Field Guide

**One-click guide for officers collecting phones**  
**Prevents data loss — keeps iPhones in AFU**

---

---

## How to Use

1. **Open the link** on any phone or tablet  
2. Tap **"Start Step-by-Step Guide"**  
3. Answer two questions:  
   - Is the phone **ON**?  
   - Do you have the **passcode**?  
4. Follow the **big green/red buttons**  
5. **Print final screen** for chain of custody

---

## Key Rules

| Situation | Action |
|---------|--------|
| **ON + Passcode** | Unlock → Airplane Mode → Bring anytime |
| **ON + NO Passcode** | **RUSH TO LAB within 72 hours** |
| **OFF** | **LEAVE OFF** — Label: “DO NOT POWER ON” |

> **iPhones auto-reboot after 72 hours without passcode → ALL DATA LOCKED**
